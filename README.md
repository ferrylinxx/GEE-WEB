Web corporativa de [Gabinet Estudis Econòmics](https://geeconomics.com) desenvolupada amb Next.js 16, Tailwind CSS 4 i TypeScript.

## Pàgines

- **Inici** - Hero slider amb presentació general
- **Presentació** - Història i reconeixements
- **Que fem** - Serveis de consultoria
- **Qui som** - Equip professional
- **Clients** - Sectors i àmbits d'actuació
- **Contacte** - Formulari funcional + Google Maps

## Tech Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- TypeScript
- Google Analytics 4

## Desenvolupament

```bash
npm install
npm run dev
```

## Desplegament

Desplegat a [Vercel](https://vercel.com). Cada push a `main` es desplega automàticament.
